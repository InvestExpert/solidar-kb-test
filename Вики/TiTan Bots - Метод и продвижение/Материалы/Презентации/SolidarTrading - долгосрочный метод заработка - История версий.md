# SolidarTrading - долгосрочный метод заработка - История версий

## Правило

Файл презентации не переименовывать в "последнюю версию" и не создавать отдельный дубль с таким названием.

Актуальная версия определяется по самой большой версии, дате файла и записи в этой истории.

Имя файла должно сохранять смысл, версию и дату.

Старые неактуальные версии убирать в общий корневой `Архив`.

В истории писать имя файла, дату и смысл изменений. Не обслуживать архивные пути.
