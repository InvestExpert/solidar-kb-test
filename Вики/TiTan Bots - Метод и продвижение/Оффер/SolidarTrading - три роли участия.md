# SolidarTrading - три роли участия

## Суть

Презентация `Где прячется доход` предлагает три роли участия. Это удобно для продажи, потому что один и тот же метод можно объяснять разным людям по-разному.

## 1. User

Для человека, который хочет использовать систему для своего счета.

Рабочий смысл:

- деньги остаются на личном счете;
- торговля автоматизируется;
- человек не тратит время на ручную торговлю;
- контроль вывода остается у клиента.

Безопасная формулировка:

```text
Вы подключаете автоматизацию к своему счету и сохраняете контроль над деньгами. Результат зависит от рынка, настроек и выбранного риска.
```

## 2. Partner

Для лидера команды, блогера, владельца аудитории или человека, который хочет строить партнерскую модель.

Рабочий смысл:

- партнер продвигает метод;
- доход строится вокруг активности клиентов и оборота;
- модель должна быть привязана к реальному использованию брокерской инфраструктуры.

Безопасная формулировка:

```text
Партнер может получать вознаграждение по условиям брокерской программы, если его клиенты реально используют торговую инфраструктуру.
```

## 3. Business / White Label

Для компаний, команд и проектов, которые хотят запускать собственную упаковку на базе технологий SolidarTrading.

Рабочий смысл:

- свой бренд;
- техническая основа на стороне SolidarTrading;
- отдельная упаковка для аудитории партнера.

Статус: `needs_proof`.

До отдельной проверки нельзя обещать запуск за 48 часов как гарантированный срок.

Безопасная формулировка:

```text
Для команд и компаний можно рассматривать отдельную упаковку метода под свой бренд. Сроки, условия и ответственность требуют отдельного согласования.
```
