# TiTan Bots - как выбирать брокера для торговых роботов

## Короткий ответ

Для TiTan Bots брокер - это не просто место, где человек открывает счет. Это инфраструктура, через которую работает торговый робот.

Продажная формулировка:

```text
Не каждый брокер подходит для робота. Красивое приложение не заменяет нормальную торговую инфраструктуру.
```

## Что база знает

- Для ручной торговли человеку может быть достаточно удобного приложения.
- Для робота важнее платформа, исполнение, спреды, комиссии, стабильность и возможность автоматизации.
- Наличие партнерки полезно для бизнеса, но не доказывает надежность брокера.
- Брокера нужно оценивать отдельно как торговую площадку, техническую среду и партнерскую инфраструктуру.
- Закрытая собственная платформа - слабое место для TiTan Bots, если на ней нельзя нормально запускать советники, подключать API или переносить стратегию.

## Основные критерии

### 1. Техническая совместимость

Для торговых роботов важны:

- MT4;
- MT5;
- cTrader;
- API или другой нормальный способ автоматизации;
- возможность запускать советники;
- доступная история для тестов;
- стабильная работа терминала.

Если брокер работает только через закрытую платформу, его нельзя считать основным кандидатом для TiTan Bots без отдельной проверки.

### 2. Торговые условия

Проверять нужно:

- спреды;
- комиссии;
- проскальзывания;
- тип исполнения;
- ограничения на советники;
- качество котировок;
- минимальный депозит;
- максимальное плечо по региону клиента.

Для клиента это можно объяснять просто:

```text
Роботу мало красивого кабинета. Ему нужна площадка, где сделки исполняются стабильно, а условия не съедают результат.
```

### 3. Юрисдикция и надежность

Проверять нужно:

- какая компания обслуживает клиента;
- какая лицензия действует сейчас;
- в какой стране открыт счет;
- какие есть ограничения для региона клиента;
- как работает ввод и вывод средств.

Стаж бренда важен, но сам по себе не доказывает надежность.

### 4. Партнерская программа

Партнерка важна для SolidarTrading и продвижения TiTan Bots, но это отдельный слой оценки.

Правило:

```text
Партнерка - плюс для бизнеса, но не доказательство надежности брокера.
```

## Как объяснять клиенту

Коротко:

```text
Мы смотрим не только на бренд брокера. Для торгового робота важнее, чтобы была нормальная платформа, стабильное исполнение, понятные комиссии и возможность автоматизации.
```

Если клиент спрашивает, почему нельзя выбрать любого брокера:

```text
Для ручной торговли брокер может быть удобным. Для торгового бота он может быть бесполезным, если не поддерживает нужную платформу или ограничивает автоматизацию.
```

## Ограничения

- Не публиковать старые или непроверенные спреды, плечо, лицензии и партнерские условия как актуальные факты.
- Не говорить, что один брокер "самый надежный", если это не подтверждено отдельной проверкой.
- Не путать выбор брокера с гарантией прибыли.
- Не подменять анализ брокера партнерской выгодой.
