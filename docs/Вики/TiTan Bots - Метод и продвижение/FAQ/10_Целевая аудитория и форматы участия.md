# 10_Целевая аудитория и форматы участия

## Назначение хаба

Кому подходит метод, кому не подходит, какие есть роли участия, чем отличаются Copy Trading, партнерство, лидерство, отдельная покупка TiTan ATLAS, White Label и крупный депозит.

Полные ответы хранятся в канонических карточках раздела `TiTan Bots - Метод и продвижение`. Этот файл хранит карту вопросов и маршруты.

Главное правило:

```text
Основной публичный формат SolidarTrading для обычного пользователя - подключение TiTan ATLAS через Copy Trading на личном брокерском счете.
Отдельная покупка TiTan ATLAS и White Label доступны по индивидуальному запросу, но не являются первым массовым оффером для новичка.
```
